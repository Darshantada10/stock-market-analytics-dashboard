# Power BI Dashboard Architecture

## Dashboard Pages

### 1. Executive Overview
Purpose:
High-level KPI and market summary dashboard.

Visuals:
- KPI cards
- Volatility trend line chart
- Top volatile companies
- Anomaly distribution

### 2. Volatility Analysis
Purpose:
Analyze rolling volatility trends across companies.

Visuals:
- Rolling volatility time-series
- Scatter plot
- Volatility comparison matrix

### 3. Price Anomaly Detection
Purpose:
Investigate abnormal market behavior.

Visuals:
- Anomaly timeline
- Conditional anomaly table
- Return distribution visuals

### 4. Market Movers
Purpose:
Identify top gainers, losers, and highest movement days.

Visuals:
- Ranking tables
- Top movers charts
- Leaderboards

### 5. Company Drillthrough
Purpose:
Detailed company-level investigation dashboard.

Visuals:
- Company price trend
- Volatility trend
- Anomaly history
- Volume analysis
