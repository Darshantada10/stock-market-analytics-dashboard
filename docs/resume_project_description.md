# Resume Project Description

## Stock Market Analytics Dashboard | SQL Server, Power BI

Built an end-to-end analytics engineering and business intelligence solution for analyzing historical NIFTY 50 stock market data using SQL Server and Power BI.

### Key Contributions

- Designed automated SQL-based ETL pipeline to consolidate 45 stock market CSV datasets into centralized analytical database.
- Processed and validated 233,000+ historical trading records covering 2000–2023 market activity.
- Developed dimensional data model using fact and dimension tables for optimized analytical querying.
- Implemented advanced SQL analytics using window functions, rolling averages, volatility calculations, and anomaly detection logic.
- Built interactive multi-page Power BI dashboard featuring KPI reporting, Top-N volatility rankings, dynamic filtering, and DAX-driven visual interactions.
- Created enterprise-style validation and performance optimization layers for analytical integrity and scalability.
